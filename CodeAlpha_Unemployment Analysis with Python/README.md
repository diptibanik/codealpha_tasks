This project analyzes unemployment rate data using Python to understand unemployment trends and patterns over time.

Key tasks performed:
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Visualization of unemployment trends
- Analysis of the impact of COVID-19 on unemployment rates
- Identification of key patterns and seasonal trends
- Generation of data-driven insights to support informed decision-making

Tools Used:
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
